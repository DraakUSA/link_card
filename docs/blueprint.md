# **App Name**: LinkCard

## Core Features:

- Website Link: Opens the user's website in a browser.
- LinkedIn Link: Opens the user's LinkedIn profile in a browser.
- GitHub Link: Opens the user's GitHub profile in a browser.
- Contact Me: Opens the device's default email app, pre-populated with the user's email address for contact.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and trustworthiness.
- Background color: Very light grey (#F5F5F5) for a clean and modern look.
- Accent color: Soft lavender (#C5CAE9) to add a touch of sophistication without overwhelming the design.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern aesthetic.
- Use simple, minimalist icons to represent each link (website, LinkedIn, GitHub, email).
- Arrange the links in a clear, single-column layout, ensuring easy access on mobile devices.
- Subtle fade-in animations on link icons upon opening the app.
